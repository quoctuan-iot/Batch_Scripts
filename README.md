5 OPEN FILE EXPLORER.

6 LOGIN B2B.

7 LOCK SCREEN.

8 LOCK SCREEN USING TIMER 120s.

9 OPEN NEW COMMAND PORT.

10 OPEN MICROSOFT EAGE.
